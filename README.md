# React Resume Builder

A simple and clean resume builder with live preview functionality.

## Features

- **Live Preview**: See your resume update in real-time as you type
- **Form Layout**: Clean form interface on the left side
- **Professional Preview**: Resume preview on the right side
- **Dynamic Sections**: Add multiple education entries and work experiences
- **Responsive Design**: Clean CSS flexbox layout

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will start on `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## Usage

1. Fill in your personal information (name and email)
2. Add your skills (comma-separated)
3. Add your education details (can add multiple entries)
4. Add your work experience (can add multiple entries)
5. See your resume preview update live on the right side

## Technologies Used

- React 18
- Vite
- CSS Flexbox
- React Hooks (useState)
